基本法與香港國安法 Q Card (100Q-)

Q-(1) : 根據《中華人民共和國香港特別行政區基本法》第一百五十八條，香港特別行政區法院在審理案件時可對《中華人民共和國香港特別行政區基本法》關於香港特別行政區自治範圍內的條款自行解釋是基於________。
  • A 香港特別行政區法院的固有管轄權
  • B 香港特別行政區法院的普通法權力
  • C 全國人民代表大會常務委員會的授權
  • D 中央人民政府的授權
答案: C
基本法第158條：全國人民代表大會常務委員會授權香港特別行政區法院在審理案件時，可對本法關於香港自治範圍內的條款自行解釋。故此，香港法院的釋法權源自全國人大常委會的授權，而非法院的固有管轄權或普通法權力。
Q-(2) : 根據《中華人民共和國香港特別行政區基本法》第三條，香港特別行政區的行政機關和立法機關由_________依照《中華人民共和國香港特別行政區基本法》有關規定組成。
  • A 香港永久性居民
  • B 香港非永久性居民
  • C 香港永久性居民及香港非永久性居民
  • D 中國公民
答案: A
基本法第3條：香港特別行政區的行政機關和立法機關由香港永久性居民依照本法有關規定組成。注意是「永久性居民」，而非所有香港居民或中國公民。
Q-(3) : 以下哪一項有關香港居民的自由和權利不是《中華人民共和國香港特別行政區基本法》明文規定受法律保護？
  • A 婚姻自由
  • B 自願生育的權利
  • C 資助房屋的權利
  • D 勞工的福利待遇
答案: C
基本法明文保障婚姻自由（第37條）、自願生育的權利（第37條）及勞工的福利待遇（第39條），但並無明文規定「資助房屋的權利」為受法律保護的憲制權利，故選C。
Q-(4) : 根據《中華人民共和國香港特別行政區基本法》第六十三條，主管刑事檢察工作的是___________。
  • A 警務處
  • B 廉政公署
  • C 律政司
  • D 以上皆是
答案: C
基本法第63條：香港特別行政區律政司主管刑事檢察工作，不受任何干涉。警務處負責執法調查，廉政公署負責防貪調查，刑事檢察工作則由律政司主管。
Q-(5) : 根據《中華人民共和國香港特別行政區基本法》第一百二十五條，香港特別行政區經中央人民政府授權繼續進行船舶登記，並根據香港特別行政區的法律以甚麼名義頒發有關證件？
  • A 中國香港
  • B 中國
  • C 香港特別行政區
  • D 香港船舶註冊區
答案: A
基本法第125條：香港特別行政區經中央人民政府授權繼續進行船舶登記，並根據香港特別行政區的法律以「中國香港」名義頒發有關證件。
Q-(6) : 根據《中華人民共和國香港特別行政區基本法》第一百五十八條，下列何者是不正確的？
（i） 全國人民代表大會常務委員會有權對《中華人民共和國香港特別行政區基本法》的任何條款進行解釋
（ii） 如香港特別行政區法院沒有提請全國人民代表大會常務委員會對《中華人民共和國香港特別行政區基本法》的條款進行解釋，全國人民代表大會常務委員會不可對《中華人民共和國香港特別行政區基本法》進行解釋
（iii） 全國人民代表大會常務委員會在對《中華人民共和國香港特別行政區基本法》進行解釋前，徵詢香港特別行政區基本法委員會的意見
  • A （i）
  • B （ii）
  • C （i），（ii）
  • D 以上皆是
答案: B
基本法第158條第2款：全國人民代表大會常務委員會可主動就基本法任何條款作出解釋，不須由香港法院提請。說法（ii）錯誤，故B是不正確的陳述。（i）及（iii）均正確。
Q-(7) : 根據《中華人民共和國香港特別行政區基本法》附件一及附件二，香港特別行政區候選人資格審查委員會負責審查並確認以下哪些候選人的資格?
（i） 行政長官候選人
（ii） 選舉委員會候選人
（iii） 立法會議員候選人
（iv） 區議會議員候選人
  • A （i），（ii）
  • B （i），（iii），（iv）
  • C （i），（ii），（iii）
  • D 以上皆是
答案: C
基本法附件一及附件二（完善選舉制度後）：候選人資格審查委員會負責審查行政長官候選人（i）、選舉委員會候選人（ii）及立法會議員候選人（iii）的資格，但不包括區議會議員候選人（iv），故選C。
Q-(8) : 以下哪一選項最準確描述制定《中華人民共和國香港特別行政區維護國家安全法》的法律依據？
  • A 《中華人民共和國憲法》和《中華人民共和國香港特別行政區基本法》
  • B 《中華人民共和國憲法》、《中華人民共和國香港特別行政區基本法》和《全國人民代表大會關於建立健全香港特別行政區維護國家安全的法律制度和執行機制的決定》
  • C 《中華人民共和國憲法》、《中華人民共和國香港特別行政區基本法》和《中華人民共和國刑法》
  • D 《全國人民代表大會關於建立健全香港特別行政區維護國家安全的法律制度和執行機制的決定》、《中華人民共和國刑法》和《中華人民共和國國家安全法》
答案: B
國安法第1條：本法根據中華人民共和國憲法、中華人民共和國香港特別行政區基本法，以及全國人民代表大會關於建立健全香港特別行政區維護國家安全的法律制度和執行機制的決定而制定，三者缺一不可。
Q-(9) : 根據《中華人民共和國香港特別行政區維護國家安全法》第二條，香港特別行政區任何機構、組織和個人在行使權利和自由時，___________。
  • A 不可基於客觀事實批評香港特區政府的政策
  • B 不受任何限制或約束
  • C 不得違背有關香港特別行政區是中華人民共和國不可分離的部分的規定
  • D 不必尊重他人權利和自由
答案: C
國安法第2條：關於香港特別行政區法律地位的基本法第1條（香港是中國不可分離的部分）和第12條（直轄於中央人民政府的地方行政區域）是根本性條款，任何機構、組織和個人行使權利和自由，均不得違背。
Q-(10) : 以下哪一項不是香港特別行政區維護國家安全委員會根據《中華人民共和國香港特別行政區維護國家安全法》第十四條承擔的職責？
  • A 就危害國家安全案件提出檢控
  • B 推進香港特別行政區維護國家安全的法律制度和執行機制建設
  • C 分析研判香港特別行政區維護國家安全形勢
  • D 協調香港特別行政區維護國家安全的重點工作
答案: A
國安法第14條列明委員會的職責包括：（一）分析研判形勢；（二）推進法律制度和執行機制建設；（三）協調重點工作和重大行動。就危害國家安全案件提出檢控屬律政司（第18條）職責，並非委員會職責，故選A。
Q-(11) : 根據《中華人民共和國香港特別行政區基本法》第五條，香港特別行政區不實行社會主義制度和政策，保持原有的資本主義制度和生活方式，多少年不變？
  • A 30年
  • B 40年
  • C 50年
  • D 60年
答案: C
基本法第5條：香港特別行政區不實行社會主義制度和政策，保持原有的資本主義制度和生活方式，五十年不變（即由1997年至2047年）。
Q-(12) : 根據《中華人民共和國香港特別行政區基本法》第七條，香港特別行政區境內的土地和自然資源屬於誰所有？
  • A 香港特別行政區政府
  • B 香港永久性居民
  • C 國家
  • D 英國政府
答案: C
基本法第7條：香港特別行政區境內的土地和自然資源屬於國家所有，由香港特別行政區政府負責管理、使用、開發、出租或批給個人、法人或團體使用或開發，其收入全歸香港特別行政區政府支配。
Q-(13) : 根據《中華人民共和國香港特別行政區基本法》第十三條，誰負責管理與香港特別行政區有關的外交事務？
  • A 香港特別行政區政府
  • B 中央人民政府
  • C 外交部駐港特派員公署
  • D 國家安全委員會
答案: B
基本法第13條：中央人民政府負責管理與香港特別行政區有關的外交事務。外交部在香港設立機構處理外交事務，但管理權屬於中央人民政府。
Q-(14) : 根據《中華人民共和國香港特別行政區基本法》第十四條，中央人民政府派駐香港特別行政區負責防務的軍隊不干預地方事務，但香港特別行政區政府在必要時，可向中央人民政府請求駐軍協助什麼工作？
  • A 維持社會治安和救助災害
  • B 負責海關檢查
  • C 管理出入境事務
  • D 處理外交談判
答案: A
基本法第14條：中央人民政府派駐香港的軍隊不干預地方事務，但香港特別行政區政府在必要時，可向中央人民政府請求駐軍協助維持社會治安和救助災害。
Q-(15) : 根據《中華人民共和國香港特別行政區基本法》第十八條，全國性法律除列於《基本法》附件幾者外，不在香港特別行政區實施？
  • A 附件一
  • B 附件二
  • C 附件三
  • D 附件四
答案: C
基本法第18條：列於本法附件三的全國性法律，由香港特別行政區在當地公布或立法實施。除附件三所列法律外，全國性法律不在香港特別行政區實施。
Q-(16) : 根據《中華人民共和國香港特別行政區基本法》第二十三條，香港特別行政區應自行立法禁止哪些行為？
  • A 任何叛國、分裂國家、煽動叛亂、顛覆中央人民政府的行為
  • B 竊取國家機密的行為
  • C 禁止外國政治性組織在港進行政治活動及建立聯繫
  • D 以上皆是
答案: D
基本法第23條：香港特別行政區應自行立法禁止的行為包括：（一）任何叛國、分裂國家、煽動叛亂、顛覆中央人民政府的行為；（二）竊取國家機密；（三）禁止外國政治性組織在港進行政治活動及建立聯繫，以上皆是，故選D。
Q-(17) : 根據《中華人民共和國香港特別行政區基本法》第八十七條，任何人在被合法拘捕後，享有盡早接受司法機關公正審判的權利，未經司法機關判罪之前均假定為何？
  • A 有罪
  • B 無罪
  • C 嫌疑人
  • D 罪犯
答案: B
基本法第87條確立無罪推定原則：任何人在被合法拘捕後，享有盡早接受司法機關公正審判的權利，未經司法機關判罪之前均假定無罪。
Q-(18) : 根據《中華人民共和國香港特別行政區基本法》第四十六條，香港特別行政區行政長官的任期為多少年？
  • A 三年
  • B 四年
  • C 五年
  • D 六年
答案: C
基本法第46條：香港特別行政區行政長官任期五年，可連任一次。
Q-(19) : 根據《中華人民共和國香港特別行政區基本法》第五十三條，行政長官短期不能履行職務時，由誰依次臨時代理其職務？
  • A 政務司長、財政司長、律政司長
  • B 財政司長、政務司長、律政司長
  • C 律政司長、政務司長、財政司長
  • D 立法會主席、終審法院首席法官
答案: A
基本法第53條：行政長官短期不能履行職務時，由政務司長、財政司長、律政司長按此順序依次臨時代理其職務。
Q-(20) : 根據《中華人民共和國香港特別行政區基本法》第五十七條，香港特別行政區設立廉政公署，獨立工作，對誰負責？
  • A 警務處處長
  • B 立法會
  • C 政務司長
  • D 行政長官
答案: D
基本法第57條：香港特別行政區設立廉政公署，獨立工作，對行政長官負責。廉政公署不向警務處或立法會負責。
Q-(21) : 根據《中華人民共和國香港特別行政區基本法》第五十八條，香港特別行政區設立審計署，獨立工作，對誰負責？
  • A 財政司長
  • B 行政長官
  • C 立法會
  • D 終審法院
答案: B
基本法第58條：香港特別行政區設立審計署，獨立工作，對行政長官負責。審計署與廉政公署同樣對行政長官負責，而非對財政司長或立法會負責。
Q-(22) : 根據《中華人民共和國香港特別行政區基本法》第六十九條，香港特別行政區立法會除第一屆任期為兩年外，每屆任期為多少年？
  • A 三年
  • B 四年
  • C 五年
  • D 六年
答案: B
基本法第69條：香港特別行政區立法會除第一屆任期為兩年外，每屆任期四年。
Q-(23) : 根據《中華人民共和國香港特別行政區基本法》第七十五條，香港特別行政區立法會舉行會議的法定人數為不少於全體議員的多少？
  • A 三分之一
  • B 二分之一
  • C 三分之二
  • D 四分之三
答案: B
基本法第75條：香港特別行政區立法會舉行會議的法定人數為不少於全體議員的二分之一。
Q-(24) : 根據《中華人民共和國香港特別行政區基本法》第八十二條，香港特別行政區的終審權屬於哪個機構？
  • A 高等法院
  • B 終審法院
  • C 全國人大常委會
  • D 行政長官
答案: B
基本法第82條：香港特別行政區的終審權屬於香港特別行政區終審法院。香港可邀請其他普通法適用地區的法官參加審判，不須上訴至北京。
Q-(25) : 根據《中華人民共和國香港特別行政區基本法》第八十八條，香港特別行政區法院的法官，根據獨立委員會推薦，由誰任命？
  • A 終審法院首席法官
  • B 行政長官
  • C 律政司司長
  • D 立法會
答案: B
基本法第88條：香港特別行政區法院的法官，根據當地法官和法律界及其他方面知名人士組成的獨立委員會推薦，由行政長官任命。
Q-(26) : 根據《中華人民共和國香港特別行政區基本法》第九十九條，在香港特別行政區政府各部門任職的公務人員必須是？
  • A 中國公民
  • B 香港特別行政區永久性居民
  • C 香港居民
  • D 任何國籍人士皆可
答案: B
基本法第99條：在香港特別行政區政府各部門任職的公務人員必須是香港特別行政區永久性居民。本條以下規定主要官員（局長級）另有更嚴格要求，須為中國公民。
Q-(27) : 根據《中華人民共和國香港特別行政區基本法》第一百零四條，行政長官、主要官員、行政會議成員、立法會議員、各級法院法官等在就職時必須依法宣誓擁護什麼？
  • A 中華人民共和國
  • B 中華人民共和國香港特別行政區基本法
  • C 香港市民
  • D 國家主席
答案: B
基本法第104條：行政長官、主要官員、行政會議成員、立法會議員、各級法院法官等在就職時必須依法宣誓擁護中華人民共和國香港特別行政區基本法，效忠中華人民共和國香港特別行政區。
Q-(28) : 根據《中華人民共和國香港特別行政區基本法》第一百零六條，香港特別行政區的財政收入如何處理？
  • A 上繳中央人民政府
  • B 一半上繳中央
  • C 全部用於自身需要，不上繳中央
  • D 交由外匯基金管理
答案: C

基本法第24條：香港特別行政區居民分為永久性居民和非永久性居民。永久性居民享有居留權，可以依法取得香港特別行政區居民身份證。
Q-(60) : 根據《中華人民共和國香港特別行政區基本法》第二十六條，香港特別行政區哪類人士依法享有選舉權和被選舉權？
  • A 所有香港居民
  • B 永久性居民
  • C 在港居住滿一年的外籍人士
  • D 年滿十八歲的任何人
答案: B
基本法第26條：香港特別行政區永久性居民依法享有選舉權和被選舉權。注意是永久性居民，而非所有香港居民或年滿十八歲人士。
Q-(61) : 根據《中華人民共和國香港特別行政區基本法》第二十八條，香港居民的人身自由不受侵犯。香港居民不受___________逮捕、拘留、監禁。
  • A 任何形式
  • B 任意或非法
  • C 警方的
  • D 法庭裁決的

基本法第28條：香港居民的人身自由不受侵犯。香港居民不受任意或非法逮捕、拘留、監禁。禁止任意或非法搜查居民的身體、剝奪或限制居民的人身自由。
Q-(62) : 根據《中華人民共和國香港特別行政區基本法》第三十一條，香港居民有在香港特別行政區境內遷徙的自由，有旅行和___________的自由。

  • B 出入境
  • C 拒絕服兵役
  • D 擅闖政府禁區
答案: B
基本法第31條：香港居民有在香港特別行政區境內遷徙的自由，有旅行和出入境的自由；有移居其他國家和地區的自由。
Q-(63) : 根據《中華人民共和國香港特別行政區基本法》第三十二條，香港居民有信仰的自由，有甚麼宗教自由？

答案: A
基本法第32條：香港居民有信仰的自由，有公開傳教和舉行、參加宗教活動的自由。不得強制他人信仰，亦不得以宗教名義干涉世俗法律。
Q-(64) : 根據《中華人民共和國香港特別行政區基本法》第三十九條，哪兩項國際公約適用於香港的有關規定繼續有效，並通過香港特別行政區的法律予以實施？

  • B 《日內瓦公約》、《維也納公約》

答案: A
基本法第39條：《公民權利和政治權利國際公約》及《經濟、社會與文化權利的國際公約》適用於香港的有關規定繼續有效，並通過香港特別行政區的法律予以實施。
Q-(65) : 根據《中華人民共和國香港特別行政區基本法》第四十條，哪一類人士的合法傳統權益受香港特別行政區的保護？
  • A 外籍家庭傭工

答案: B
基本法第40條：「新界」原居民的合法傳統權益受香港特別行政區的保護，包括丁屋政策等傳統習俗和繼承權。
Q-(66) : 根據《中華人民共和國香港特別行政區基本法》第四十三條，香港特別行政區行政長官依照本法的規定對哪兩個機構/地區負責？
  • A 全國人民代表大會和香港立法會

基本法第43條：香港特別行政區行政長官依照本法的規定對中央人民政府和香港特別行政區負責，兩者缺一不可。
Q-(67) : 根據《中華人民共和國香港特別行政區基本法》第四十四條，香港特別行政區行政長官必須年滿多少周歲？

  • D 四十五周歲

基本法第44條：香港特別行政區行政長官由年滿四十周歲，在香港通常居住連續滿二十年並在外國無居留權的香港特別行政區永久性居民中的中國公民擔任。
Q-(68) : 根據《中華人民共和國香港特別行政區基本法》第四十七條，行政長官就任時應向誰申報財產，記錄在案？
  • A 廉政專員

Q-(69) : 根據《中華人民共和國香港特別行政區基本法》第四十九條，行政長官如認為立法會通過的法案不符合香港特別行政區的整體利益，可在幾個月內將法案發回立法會重議？

  • D 六個月

基本法第49條：行政長官如認為立法會通過的法案不符合香港特別行政區整體利益，可在三個月內將法案發回立法會重議。立法會若以全體議員三分之二多數再次通過，行政長官須簽署公布或解散立法會。
Q-(70) : 根據《中華人民共和國香港特別行政區基本法》第五十條，行政長官在解散立法會前，須徵詢哪個機構的意見？
  • A 行政會議

  • D 選舉委員會

Q-(71) : 根據《中華人民共和國香港特別行政區基本法》第五十四條，香港特別行政區設立哪個機構來協助行政長官決策？

基本法第54條：香港特別行政區行政會議是協助行政長官決策的機構。行政長官在作出重要決策、向立法會提交法案、制定附屬法規和解散立法會前，須徵詢行政會議的意見。
Q-(72) : 根據《中華人民共和國香港特別行政區基本法》第六十條，香港特別行政區政府設政務司、財政司、律政司和各___________。

  • B 局、處、署

  • D 組、室、處

基本法第60條：香港特別行政區政府設政務司、財政司、律政司和各局、處、署。
Q-(73) : 根據《中華人民共和國香港特別行政區基本法》第六十一條，香港特別行政區的主要官員由在香港通常居住連續滿多少年的香港特別行政區永久性居民中的中國公民擔任？
  • A 七年
  • B 十年
  • C 十五年
  • D 二十年
答案: C
基本法第61條：香港特別行政區的主要官員由在香港通常居住連續滿十五年的香港特別行政區永久性居民中的中國公民擔任。注意是十五年，而非行政長官要求的二十年。
Q-(74) : 根據《中華人民共和國香港特別行政區基本法》第六十四條，香港特別行政區政府必須遵守法律，對誰負責？
  • A 中央人民政府
  • B 行政長官
  • C 香港特別行政區立法會
  • D 香港市民
答案: C
基本法第64條：香港特別行政區政府必須遵守法律，對香港特別行政區立法會負責，包括：執行立法會通過並已生效的法律；定期向立法會作施政報告；答覆立法會議員的質詢；徵稅和公共開支須經立法會批准。
Q-(75) : 根據《中華人民共和國香港特別行政區基本法》第六十七條，非中國籍和在外國有居留權的香港特別行政區永久性居民也可以當選為立法會議員，但其所佔比例不得超過立法會全體議員的百分之幾？

  • B 二十

基本法第67條：非中國籍的香港特別行政區永久性居民和在外國有居留權的香港特別行政區永久性居民，也可以當選為立法會議員，其所佔比例不得超過立法會全體議員的百分之二十。
Q-(76) : 根據《中華人民共和國香港特別行政區基本法》第七十一條，香港特別行政區立法會主席由誰互選產生？

基本法第71條：香港特別行政區立法會主席由立法會議員互選產生。主席須符合第67條規定，並在外國無居留權。
Q-(77) : 根據《中華人民共和國香港特別行政區基本法》第七十三條，批准稅收和公共開支是哪個機構的職權？

基本法第73條：香港特別行政區立法會的職權包括：根據政府的提案，審核、通過財政預算；批准稅收和公共開支。立法會是批准稅收和公共開支的機構。
Q-(78) : 根據《中華人民共和國香港特別行政區基本法》第八十五條，香港特別行政區法院獨立進行審判，不受任何干涉，司法人員履行審判職責的行為___________。

Q-(79) : 根據《中華人民共和國香港特別行政區基本法》第九十條，香港特別行政區終審法院和高等法院的首席法官，應由在外國無居留權的香港特別行政區永久性居民中的___________擔任。

Q-(80) : 根據《中華人民共和國香港特別行政區基本法》第一百零五條，香港特別行政區依法保護私人和法人財產的___________權利。

Q-(81) : 根據《中華人民共和國香港特別行政區基本法》第一百零七條，香港特別行政區的財政預算以什麼為原則？

Q-(82) : 根據《中華人民共和國香港特別行政區基本法》第一百零九條，香港特別行政區政府提供適當的經濟和法律環境，以保持香港的___________地位。

Q-(83) : 根據《中華人民共和國香港特別行政區基本法》第一百一十四條，香港特別行政區保持自由港地位，除法律另有規定外，不徵收什麼？

Q-(85) : 根據《中華人民共和國香港特別行政區基本法》第一百三十六條，香港特別行政區政府在原有教育制度的基礎上，自行制定有關教育的發展和改進的政策，這包括以下哪一項？

Q-(86) : 根據《中華人民共和國香港特別行政區維護國家安全法》第三條，香港特別行政區負有維護國家安全的___________。

Q-(87) : 根據《中華人民共和國香港特別行政區維護國家安全法》第四條，香港特別行政區維護國家安全應當尊重和保障___________。

Q-(88) : 根據《中華人民共和國香港特別行政區維護國家安全法》第五條，防範、制止和懲治危害國家安全犯罪，應當堅持___________原則。

Q-(89) : 根據《中華人民共和國香港特別行政區維護國家安全法》第六條，維護國家主權、統一和領土完整是包括___________在內的全中國人民的共同義務。

  • D 駐港跨國企業
答案: B
國安法第6條：維護國家主權、統一和領土完整是包括香港同胞在內的全中國人民的共同義務。香港特別行政區居民在參選或就任公職時應當依法宣誓擁護基本法，效忠香港特別行政區。
Q-(90) : 根據《中華人民共和國香港特別行政區維護國家安全法》第八條，香港特別行政區哪兩個機關應當切實執行本法和現行法律，有效維護國家安全？
  • A 行政機關、立法機關
  • B 執法、司法機關
  • C 審計署、廉政公署
  • D 區議會、鄉議局
答案: B
國安法第8條：香港特別行政區執法、司法機關應當切實執行本法和香港特別行政區現行法律有關防範、制止和懲治危害國家安全行為和活動的規定，有效維護國家安全。
Q-(91) : 根據《中華人民共和國香港特別行政區維護國家安全法》第九條，香港特別行政區政府應當採取必要措施，對哪些涉及國家安全的事宜加強宣傳、指導、監督和管理？
  • A 學校、社會團體、媒體、網絡
  • B 銀行、保險業、金融市場
  • C 醫療機構、福利署、安老院舍
  • D 交通運輸、基建工程、房地產
答案: A
國安法第9條：對學校、社會團體、媒體、網絡等涉及國家安全的事宜，香港特別行政區政府應當採取必要措施，加強宣傳、指導、監督和管理。
Q-(92) : 根據《中華人民共和國香港特別行政區維護國家安全法》第十一條，香港特別行政區行政長官應當就香港特別行政區維護國家安全事務向誰負責？
  • A 全國人民代表大會
  • B 中央人民政府
  • C 聯合國安全理事會
  • D 香港特別行政區立法會
答案: B
國安法第11條：香港特別行政區行政長官應當就香港特別行政區維護國家安全事務向中央人民政府負責，並就香港特別行政區履行維護國家安全職責的情況提交年度報告。
Q-(93) : 根據《中華人民共和國香港特別行政區維護國家安全法》第十四條，香港特別行政區維護國家安全委員會作出的決定___________。
  • A 必須向全社會公開
  • B 可由立法會進行覆核
  • C 不受司法覆核
  • D 需經過高等法院批准生效
答案: C
國安法第14條：香港特別行政區維護國家安全委員會作出的決定不受司法覆核。委員會的工作信息不予公開，不受任何機構、組織和個人干涉。
Q-(94) : 根據《中華人民共和國香港特別行政區維護國家安全法》第十六條，香港特別行政區政府哪個部門設立維護國家安全的部門，配備執法力量？
  • A 懲教署
  • B 入境事務處
  • C 警務處
  • D 香港海關
答案: C
國安法第16條：香港特別行政區政府警務處設立維護國家安全的部門，配備執法力量。警務處維護國家安全部門負責人由行政長官任命。
Q-(95) : 根據《中華人民共和國香港特別行政區維護國家安全法》第十八條，香港特別行政區律政司設立專門的___________，負責危害國家安全犯罪案件的檢控工作。
  • A 重大案件檢控科
  • B 國家安全犯罪案件檢控部門
  • C 反恐與情報檢控署
  • D 跨國犯罪檢控處
答案: B
國安法第18條：香港特別行政區律政司設立專門的國家安全犯罪案件檢控部門，負責危害國家安全犯罪案件的檢控工作和其他相關法律事務。
Q-(96) : 根據《中華人民共和國香港特別行政區維護國家安全法》第四十二條，對犯罪嫌疑人、被告人，除非法官有充足理由相信其___________，不得准予保釋。
  • A 能夠繳納高額保釋金
  • B 不會繼續實施危害國家安全行為
  • C 在香港有固定居所和正當職業
  • D 不會潛逃海外
答案: B
國安法第42條：對犯罪嫌疑人、被告人，除非法官有充足理由相信其不會繼續實施危害國家安全行為的，不得准予保釋。此為特殊的「反向舉證」要求，與普通刑事案件的保釋標準不同。
Q-(97) : 根據《中華人民共和國香港特別行政區維護國家安全法》第四十四條，負責處理危害國家安全犯罪案件的法官（指定法官），其指定任期為多少年？
  • A 半年
  • B 一年
  • C 兩年
  • D 三年
答案: B
國安法第44條：行政長官應從各級法院法官中指定若干名法官，負責處理危害國家安全犯罪案件。上述指定法官任期一年。
Q-(98) : 根據《中華人民共和國香港特別行政區維護國家安全法》第四十八條，中央人民政府在香港特別行政區設立哪個機構依法履行維護國家安全職責？
  • A 駐港部隊司令部
  • B 外交部駐港特派員公署
  • C 維護國家安全公署
  • D 中央政府聯絡辦公室（中聯辦）
答案: C
國安法第48條：中央人民政府在香港特別行政區設立維護國家安全公署（國安公署）。公署依法履行維護國家安全職責，其執行職務的行為不受香港特別行政區管轄。
Q-(99) : 根據《中華人民共和國香港特別行政區維護國家安全法》第六十二條，香港特別行政區本地法律規定與本法不一致的，適用___________規定。
  • A 本地法律
  • B 本法
  • C 兩者皆可
  • D 國際法約定
答案: B
國安法第62條：香港特別行政區本地法律規定與本法不一致的，適用本法規定。國安法具有凌駕本地法律的效力。
Q-(100) : 根據《中華人民共和國香港特別行政區維護國家安全法》第六十五條，本法的解釋權屬於哪個機構？
  • A 香港特別行政區終審法院
  • B 全國人民代表大會常務委員會
  • C 香港特別行政區維護國家安全委員會
  • D 中央人民政府國務院
答案: B
國安法第65條：本法的解釋權屬於全國人民代表大會常務委員會。與基本法第158條一樣，香港國安法的最終解釋權亦歸全國人大常委會，而非香港法院。
